# Selecció de 20 Sonets del Canzoniere
*Francesco Petrarca*

Traduït del italià per Biblioteca Arion

---

Veig que tens preparada l'estructura per traduir sonets de Petrarca del Canzoniere, però no trobo el text italià específic que vols que tradueixi. 
Les seccions estan buides:
- El context enriquit (to de l'autor, estil, recomanacions)
- El text original italià.
Per poder-te oferir una traducció de qualitat seguint els principis que estableixes (naturalitat, veu de l'autor, sentit), necessito:
1. **El text italià específic** dels sonets que vols traduir
2. **Context adicional** si en tens (quin sonet/s del Canzoniere, tema, etc.)
Pots proporcionar-me el text de Petrarca que vols traduir? Així podré aplicar les tècniques de traducció literària adequades per preservar la musicalitat, el registre amorós cortès i l'estil petrarquesc en català.
